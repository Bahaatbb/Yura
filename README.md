## Demo Development

Clone and install.

```bash
git clone git@github.com:Bahaatbb/Yura.git
cd Yura/api
yarn install
cd ../client
yarn install
```

Run a development server.

```bash
cd Yura/api && yarn dev
```

Run a development client.

```bash
cd Yura/client && yarn dev
```
